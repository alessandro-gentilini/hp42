2 Introduction

This org-mode file collects together a handful of mathematical stuff I find useful. I keep this stuff in my custom menu.

3 Arithmetic Types

Many of the simple math functions here work on what I call an "arithmetic type" – that is a thing the 42s can do arithmetic with: real numbers, complex numbers, real matrices, & complex matrices. In order to simplify the code the ATYP function spits out a small integer that can be used in a computed GOTO to handle these four special cases.

3.1 ATYP: Return type of object

@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@ (ATYP)
@@@@ DSC: Arithmetic Type
@@@@ IN:  X: an object
@@@@ OUT: X: 1 if input X was a real number
@@@@         2 if input X was a real matrix
@@@@         3 if input X was a complex number
@@@@         4 if input X was a complex matrix
@@@@         0 if none of the above are true
@@@@ TST: free42_3.0
