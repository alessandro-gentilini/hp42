2 Introduction

This org-mode file collects together a handful of simple mathematical functions I find useful. Many of these functions work on what I call an "arithmetic type" – that is a thing the 42s can do arithmetic with: real numbers, complex numbers, real matrices, & complex matrices. In order to simplify the code the ATYP function spits out a small integer that can be used in a computed GOTO to handle these four special cases.

Function Description Works On
