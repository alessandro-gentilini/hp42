* =SGN=: Sign function

#+BEGIN_SRC hp42s :tangle no
@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@ (ref:SGN)
@@@@ DSC: Sign function
@@@@ IN:  X: a number
@@@@ OUT: X:  0 when input was 0
@@@@         -1 when input was negative
@@@@          1 when input was positive
@@@@ UPD: 2021-02-26
@@@@ TST: free42_3.0
LBL "SGN"
FUNC 11              @@## REQ:free42>=2.5.24
L4STK                @@## REQ:free42>=3.0
ENTER
ABS
X≠0?
GTO 01
@@@@ zero
0
RTN
LBL 01
@@@@ Non-zero
X<>Y
REAL?
GTO 02
@@@@ Complex non-zero
X<>Y
÷
RTN
LBL 02
@@@@ Real non-zero
X<0?
GTO 03
@@@@ Real positive
1
RTN
LBL 03
@@@@ Real negative
-1
RTN
END
#+END_SRC
